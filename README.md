# TypeScript Skills

Collection skills for TypeScript.

## Skills

### [typescript](typescript)

Typescript API skills for enterprise-grade development.

Sources:

- [Mastering TypeScript Skill](https://github.com/SpillwaveSolutions/mastering-typescript-skill/tree/main/mastering-typescript)

Installation:

```shell
npx skills add iwyfaf-ai-skills/typescript --skill typescript
```

or update:

```shell
npx skills update typescript
```

hard update:

```shell
npx skills add iwyfaf-ai-skills/typescript --skill typescript
```
